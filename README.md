# 🧵 Handmade Store - E-commerce Website

A modern and responsive **handmade e-commerce website** built using HTML, CSS, and JavaScript.  
This project simulates a real online store for handmade products with a clean UI and user-friendly experience.

---

## ✨ Features

- 🛍 Product listing page (cards layout)
- 🔍 Search bar for filtering products
- 🛒 Add to cart functionality (JavaScript)
- 💳 Checkout page UI
- 📄 About page with brand story
- 🧵 Services page (custom orders, gifts, etc.)
- 📱 Fully responsive design (mobile + desktop)
- 🎨 Modern UI with consistent branding

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript (Vanilla JS)
